# Entity Sentiment Analysis with Tweets 
## Data 
medium size data from [kaggle](https://www.kaggle.com/datasets/jp797498e/twitter-entity-sentiment-analysis/data)

## Techniques 
### Preprocessing 
- Data Cleaning
- Text Processing
  - Stop-word Removal
  - Lemmatization
  - Tf-idf Tokenization
- Categorical Data Processing
  - One-Hot Cncoding
  - Label Encoding 

### Training 
- Logistic Regression
- Support Vector Machine (With Linear Kernel) - LinearSVC

### Evaluation 
- Accuracy
- Confusion Metric
- AUC-ROC

### Library
- Panda
- sklearn
- spaCy
- matplotlib
