# Entity Sentiment Analysis with Tweets 
## Data 
medium size data from [kaggle](https://www.kaggle.com/datasets/jp797498e/twitter-entity-sentiment-analysis/data)

## Techniques 
### Preprocessing 
- stop-word removal
- lemmatization
- Tf-idftokenization

### Training 
- Logistic Regression

### Library
- Panda
- sklearn
- spaCy
